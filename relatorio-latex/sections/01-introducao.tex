\section{Introdução}

Este relatório registra, de forma objetiva, o que já foi construído e o que foi corrigido na revisão local do Trabalho 2 da MC714.
O projeto foi estruturado como uma implementação acadêmica de sistemas distribuídos com comunicação HTTP/JSON real entre três nós e um observador externo.

A revisão recente teve dois focos complementares.
O primeiro foi técnico: alinhar o código com a semântica esperada para o relógio de Lamport, a exclusão mútua de Ricart-Agrawala e a eleição de líder com Bully.
O segundo foi documental: aumentar a rastreabilidade entre teoria, código, testes, smoke tests, experimentos e resultados versionados.

O resultado é uma base local mais clara para avaliação acadêmica, ainda sem extrapolar o escopo original do trabalho.
